# CODSOFT Web Development Internship

This repository contains the tasks completed as part of the CodSoft Web Development Internship.

## Level 1 Tasks

### Task 1 - Portfolio Website
A personal portfolio website created using HTML and CSS.

- Projects Section
- Resume Section
- Contact Information

### Task 2 - Landing Page
A responsive landing page designed using HTML and CSS.
Features:
- Attractive Layout
- Header and Footer
- Multiple Sections
- Responsive Design

### Task 3 - Calculator
A basic calculator developed using HTML, CSS, and JavaScript.
Features:
- Addition
- Subtraction
- Multiplication
- Division
- User-Friendly Interface
